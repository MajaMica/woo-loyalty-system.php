🔥 WooCommerce Loyalty System 

This is a custom WooCommerce loyalty system originally developed as a functions.php implementation for a specific client website with custom business requirements.

The system introduces a simple “Vatrice” (points) model that rewards customers for purchases and allows them to redeem points as discount coupons.

⚙️ Features
Earn loyalty points (“Vatrice”) on completed orders
Dynamic points calculation based on product price thresholds
Display points in:
Cart
Customer account (shortcode)
Thank you page
Redeem points into WooCommerce discount coupons
Automatic coupon generation based on user balance
Reset points after redemption
🧩 Implementation

The code was originally implemented inside WordPress functions.php and is designed for a specific WooCommerce setup.

With minor refactoring, it can be:

moved into a standalone WooCommerce plugin
adapted for any WooCommerce store
extended with checkout integration or advanced redemption rules
