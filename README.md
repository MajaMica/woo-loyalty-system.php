# WooCommerce Loyalty System (Vatrice)

Adds points-based loyalty: 50 points for orders under 5000 RSD, 100 for above.  
Convert points to coupons (1 point = 1 RSD).

## Shortcodes
- `[order_statistics]` – show user's points
- `[fire_coupon_generator]` – form to convert points to coupon

## Installation
Upload to `/wp-content/plugins/` and activate.
